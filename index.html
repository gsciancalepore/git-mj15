Working Directory

El área de trabajo,basicamente consiste en la carpeta donde inicializamos el repositorio y donde vamos a estar creando, editando y eliminando los archivos de nuestro proyecto. Muy rar vez vamos a estar utilizando la linea de comandos en esta área.

  • dir || ls muestra el contenido de la carpeta
  • mkdir <name> crea una carpeta
  • cd <path> abre un directorio
  • git init inicializa un repositorio de git
  • rm -rf .git elimina el repositorio
